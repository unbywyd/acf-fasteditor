Plugin allows you to edit ACF fields on the frontend side

== Description ==

This plugin searches for ACF fields on the post page and allows you to quickly edit them
Features of this plugin:

- Ability to edit all ACF post fields
- Searches for widgets on the page and adds a button to them for quick editing

Searches the page for elements with the following parameters: 

- Element selector: .mixcode_widget[data-pid], .acf-widget[data-pid], .acf_widget[data-pid]
- Dataset attributes: 
-- data-pid (Post id of ACF fields)
-- data-acf-field (Specific ACF field) if not specified, all fields will be taken
